# Presentation

`agenttiohjelmointi_esitys.tex` — Beamer slides for the agent-assisted programming talk.

## Dependencies

- A TeX Live installation with `pdflatex`
- Python package `pygments` (required by the `minted` package for syntax highlighting)

```sh
pip install pygments
```

## Compiling

```sh
make
```

Output: `agenttiohjelmointi_esitys.pdf`
